LATAM - API de Consulta de Vuelos (FastAPI)

Servicio web tipo API REST para consultar disponibilidad de vuelos (origen, destino, fecha y asientos).  
Proyecto orientado a migración/despliegue en la nube usando tecnología open source.

Tecnologías
- Python 3
- FastAPI
- Uvicorn
- Pydantic

Requisitos
- Python 3.10+ (recomendado 3.11)
- pip
